# Trip_Buddy

It allow users to search online for travel companions for their desired destination. User can create new travel group and invite friends.

## Features

- **Personalized Trip Planner**
- **Budget Tracker**
- **Trip Sharing**
- **Real-Time Updates**

![Trip_Buddy Logo](ui/public/images/trip3.PNG)

## Technologies Used

- **Frontend**: React.js
- **Backend**: Node.js
- **Database**: MongoDB
