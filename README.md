# Trip_Buddy

Trip_Buddy is your ultimate travel companion that helps you plan, organize, and share your trips effortlessly.

## Features

- **Personalized Trip Planner**
- **Budget Tracker**
- **Trip Sharing**
- **Real-Time Updates**
